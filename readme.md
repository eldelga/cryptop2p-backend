## Installation

clon repository

```sh
git clone https://github.com/eldelga/cryptop2p-backend.git
```

Build image

```sh
docker build -t cryptop2p-backend:v1 .
```

Run container

```sh
docker run -p 80:8080 cryptop2p-backend:v1 
```
